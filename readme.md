ASCII Painter
=============

It's work-in-progress. Inspired by asciiflow.com, this tool converts call graph:

```
ClassA::func_1
  ClassB::func_2
    ClassD::func_4
  ClassC::func_3
```

to UML sequence:

```
    ┌───────────────────┐    ┌───────────────────┐    ┌───────────────────┐    ┌───────────────────┐
    │ClassA             │    │ClassD             │    │ClassB             │    │ClassC             │
    │                   │    │                   │    │                   │    │                   │
    │                   │    │                   │    │                   │    │                   │
    │                   │    │                   │    │                   │    │                   │
    └───────────────────┘    └───────────────────┘    └───────────────────┘    └───────────────────┘
              │                        │                        │                        │
 func_1       │                        │                        │                        │
─────────────►│                        │                        │                        │
              │                        │                        │                        │
              │ func_2                 │                        │                        │
              │────────────────────────│───────────────────────►│                        │
              │                        │                        │                        │
              │                        │  func_4                │                        │
              │                        │◄───────────────────────│                        │
              │                        │                        │                        │
              │ func_3                 │                        │                        │
              │────────────────────────│────────────────────────│───────────────────────►│
              │                        │                        │                        │
              │                        │                        │                        │
```
